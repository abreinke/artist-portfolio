Night Insects

Night Insects is a code project which was inspired by my childhood memories of two things: butterfly paintings and sights of nocturnal insects. I'm hoping that this website will help people recreate similar, visual memories.
Instructions for Creating the Drawing
  1. The first color is red. Hold down the mouse and move it anywhere on the grey canvas to start drawing.
  2. Press the Y key for yellow.
  3. Press the B key for blue.
  4. Press the D key to go back to red.
  5. Press the X key whenever you're ready to save your current drawing.
